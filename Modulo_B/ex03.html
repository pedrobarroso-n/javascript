Aritmeticos

  • (+) somar
  • (-) subtrair
  • (*) multiplicar
  • (/) dividir
  • (%) resto da divisao
  • (**) potenciaçao

    • ordem de precedencia: () > ** > % e / e * > + e -

Atribuicao:

  • x = x + 1 -> x += 1
  • x = x + 1 -> x++

    • x = 5
    x +=4
    x-= 2
    x = 7

    funciona com quase todos os operadores aritimeticos, e possivel fazer na mesma variavel

Relacionais:

  • > maior q
  • < menor q
  • >= maior igual a
  • <= menor igual a
  • == igual
  • != diferente

    • resultado sera um valor bolleano (True/False)

  • === igual tanto no tipo quanto no conteudo
    ex: 5 == 5 (true) 5 == '5' (true) //>// 5 === 5 (true) 5 === '5' (false)

Logicos:

  • ! negacao !T = F / !F = T
  • && conjucao T+T=T / T+F=F / F+T=F / F+F=F
  • || disjuncao T+T=T / T+F=T / F+T=T / F+F=F

    • aritmeticos > relacionais > ! > && > ||
